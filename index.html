Sobre o projeto

Para esse projeto pensei em desenvolver algo que parecesse simples, mas que ao mesmo tempo se tornasse complicado na medida em que eu adicionasse diferentes elementos estéticos e com funções. Sendo assim, iniciei pela construção da estrutura do html, que se mostrou curto e simples. Ademias, usei CSS e Javascript para o desenvolvimento do comportamento do design e dos inputs que funcionassem ao click, sendo os elementos que se mostraram demorados de serem resolvidos.
